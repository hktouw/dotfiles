# dotfiles & keyboard setup
```
1. Clone Repo
2. Copy all .files to ~/
3. git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
4. Copy color scheme to ~/.vim/colors
5. brew install tmux
6. VundleInstall (in vim)
7. System Preferences -> keyboard -> modifier keys -> caps lock to ctrl
```
